# distribuidos_chat

Initial:
===========
* `virtualenv venv`
* `source venv/bin/activate`
* `pip3 install -r requirements.txt`
* `python3 models.py db init`
* `python3 models.py db migrate`
* `python3 models.py db upgrade`

How to run
==============
* `python3 app.py`
* `In the browser go to http://127.0.0.1:5000/home`
